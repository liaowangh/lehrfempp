Detailed Description

template<typename SCALAR>
class lf::uscalfe::FeSpaceLagrangeO1< SCALAR >

(Bi)Linear Lagrangian Finite Element space

Just a specialization of UniformScalarFESpace based on FeLagrangeO1Tria, FeLagrangeO1Quad and FeLagrangePoint.

Example

auto mesh_factory = std::make_unique<mesh::hybrid2d::MeshFactory>(2);
auto gmsh_reader = io::GmshReader(std::move(mesh_factory), "mesh.msh");
auto fe_space =
std::make_shared<FeSpaceLagrangeO1<double>>(gmsh_reader.mesh());
// #dofs = #points in mesh:
assert(fe_space->LocGlobMap().NumDofs() == gmsh_reader.mesh()->NumEntities(2));
